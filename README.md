# Sweet Shop Management System

A simple backend system to manage a sweet shop using **Python** with **Test-Driven Development (TDD)**.

---

##  Features Implemented

- Add Sweets  
- Delete Sweets  
- View All Sweets  
- Search by Name  
- Search by Category  
- Search by Price Range  
- Purchase Sweets (with stock check & validation)  
- Restock Sweets
